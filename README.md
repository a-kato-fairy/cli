a5doc
=====

ドキュメントの作成をサポートするためのツール。

こんなことをやりたいと思ってます。
* 用語の管理とスペルチェック
* 自動作成（自動化できそうな一部）

## 用語

ドキュメント内にある用語を集めて、各所で活用する。
* スペルチェックに使う
* 索引ページを作成する

### 用語として管理する項目

* 用語名称  
* 用語分類  
    - テーブル
    - カラム
    - 画面
* 用語タイプ  
    - テーブル物理名
    - テーブル論理名
    - カラム物理名
    - カラム論理名
    - 画面項目ラベル
    - 画面名
    - 画面ID
* 記述箇所
